# 📞 AI IVR Call Assistant (Speech-to-Decision System)

An **AI-powered IVR (Interactive Voice Response) system** that listens to a caller’s speech, understands intent using AI, takes an automated decision (forward / reject / ask more), and responds with a **clear customer-care style voice**, along with **WhatsApp notifications** for the owner.

This project simulates a **real-world call center / customer support IVR flow** using Python and FastAPI.

---

## 🚀 Key Features

- 🎙️ **Speech-to-Text (STT)** using OpenAI Whisper (offline & free)
- 🧠 **AI Decision Engine** to classify calls (Important / Spam / Unclear)
- 🗣️ **Hindi IVR Voice Response** (customer-care style)
- 📩 **WhatsApp Notification** with call summary (Twilio Sandbox)
- 🌐 **FastAPI Backend** (IVR-ready API)
- 🔐 Secure environment variable handling edit
- (`.env`)
